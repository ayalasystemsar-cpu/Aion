# AION YAROKU 🛡️
*Sistema Táctico de Comando y Logística para Seguridad Privada*

Aion Yaroku es una plataforma integral de alto rendimiento diseñada para centralizar, auditar y optimizar las operaciones de agencias de seguridad en tiempo real. Construido sobre una arquitectura de nube de alta disponibilidad, el sistema elimina la fragmentación operativa unificando el control de flotas y objetivos bajo un solo centro de mando.

### ⚙️ ARQUITECTURA TÁCTICA Y FUNCIONES PRINCIPALES

* *Geolocalización Balística:* Filtro de coordenadas de alta precisión con ruteo dual integrado (Google Maps / Waze) para despliegue inmediato de supervisores.
* *Control de Presentismo QR:* Sistema de escaneo encriptado con impacto instantáneo en la base de datos central.
* *Red de Alerta S.O.S (Pánico):* Botón de emergencia con latido de sistema (TTL) para bloqueo de pantallas y aviso visual en la Central de Monitoreo.
* *Mensajería Limpia Bidireccional:* Comunicación estructurada sin duplicación de registros y con acuse de lectura integrado.
* *Gestión Documental:* Auditoría de actas, kilometraje de flotas vehiculares y control de novedades de servicio.
* *Núcleo Anti-Reseteo:* Arquitectura de estado de sesión persistente para evitar pérdidas de conexión durante la operación en terreno.
